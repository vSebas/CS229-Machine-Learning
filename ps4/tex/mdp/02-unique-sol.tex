\begin{answer}
    
Assuming $B$ has two fixed points $V_1$ and $V_2$, then
\[
B(V_1) = V_1, \qquad B(V_2) = V_2
\]
Using the contraction property,
\[
\|V_1 - V_2\|_\infty 
= \|B(V_1) - B(V_2)\|_\infty 
\le \gamma \|V_1 - V_2\|_\infty.
\]
Thus
\[
(1 - \gamma)\|V_1 - V_2\|_\infty \le 0.
\]
Since $0 < \gamma < 1$, we have $1-\gamma>0$, so the only way this can hold is if
\[
\|V_1 - V_2\|_\infty = 0,
\]
So,
\[
\boxed{V_1 = V_2}
\]
Thus, Bellman equations have at most one solution

\end{answer}
